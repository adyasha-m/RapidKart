## 👥 User Metrics

### Daily Active Users (DAU)

Number of unique users performing ≥1 event per day.

`DAU = COUNT(DISTINCT user_id)`

### Weekly Active Users (WAU)

Unique users active within a week.

### Activation Rate

Users placing first order within 24h of signup.

### Activation Rate =

Activated Users / New Users

## ⏱ Engagement Metrics

### Sessions per User

Average number of sessions per user.

Total Sessions / Active Users

### Avg Events per Session

Measures interaction depth.

## 🛒 Funnel Metrics

### Add-to-Cart Rate

Users who added to cart / Users who viewed product

### Checkout Rate

Checkout Starts / Add to Cart

### Conversion Rate (Session)

Sessions with order / Total sessions

## 🔁 Retention Metrics

### Week 1 Retention

Users active in Week 1
÷ Users active in Week 0

### Churn Rate (30-day)

Users inactive for 30 days / Total users

## 🚚 Operational Metrics

### Average Delivery Time

AVG(delivery_time_minutes)

### Late Delivery Rate

Deliveries > 30 mins / Total deliveries

## 💰 Revenue Metrics

### Average Order Value (AOV)

Total Revenue / Total Orders

### Repeat Purchase Rate

Users with >1 order / Users with ≥1 order
